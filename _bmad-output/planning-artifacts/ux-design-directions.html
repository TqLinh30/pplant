Pplant UX Design Directions

Eight mobile directions for calm student-life planning.

These explorations translate the validated PRD and DESIGN.md into concrete mobile approaches. Each uses white canvas, dark ink, restrained controls, hairline boundaries, and signature color only when it carries meaning.

0 shortlisted

Recommended base direction

Direction 1, Calm Daily Checkpoint, should be the primary app direction. It best supports the product promise: one calm overview where students understand money, time, attention, reminders, and capture without opening separate dashboards.

What to compare

  • Layout intuitiveness: does the screen answer what needs attention today?
  • Interaction style: can capture start quickly and recover gracefully?
  • Visual weight: does the interface feel mature without becoming heavy?
  • Brand alignment: are signature colors meaningful, sparse, and calm?
